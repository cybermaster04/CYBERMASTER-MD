📖 About

**CYBERMASTER-MD** is a high-performance WhatsApp bot built with `@whiskeysockets/baileys`. Uses SESSION_ID authentication, SQLite for data persistence, and a modular plugin system.

**Why CYBERMASTER-MD?**
- **Fast**: Optimized event handling + SQLite caching
- **Stable**: Auto-reconnect, error handling, memory management  
- **Modular**: Drop new commands in `/plugins` folder
- **Secure**: Non-root Docker, owner-only commands, LID resolution

🛠️ Local Installation

Requirements
- Node.js v18+
- Git
- FFmpeg for media processing

Setup
```bash
Clone repository
git clone https://github.com/cybermaster04/CYBERMASTER-MD.git
cd CYBERMASTER-MD

Install dependencies
npm install

Configure environment
cat > .env << EOF
SESSION_ID=CYBERMASTER:~your_base64_session
PREFIX=.
OWNER_NUMBER=2637xxxxxxx
BOT_NAME=CYBERMASTER-MD
MODE=public
EOF

Start bot
npm start
